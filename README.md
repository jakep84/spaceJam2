SpaceJam is a 3D visualization application built using Cesium, a JavaScript library for creating 3D globes and 2D maps in a web browser without a plugin. It uses WebGL for hardware-accelerated graphics, and is cross-platform, cross-browser, and tuned for dynamic-data visualization.

http://cesiumjs.org/

### Get Started ###

Visit the [Downloads page](http://cesiumjs.org/downloads.html) or use the npm module:
```
npm install cesium
```

Have questions?  Ask them on the [forum](http://cesiumjs.org/forum.html).

Interested in contributing?  See [CONTRIBUTING.md](CONTRIBUTING.md).
